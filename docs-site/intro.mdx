## What is HyperClaw?

HyperClaw is a **local-first AI agent** that runs on your machine. It connects to:

- **Messaging channels** — Telegram, Discord, Slack, WhatsApp, and more
- **Webhooks** — inbound HTTP for cron, Zapier, external services
- **CLI** — `hyperclaw run`, `hyperclaw doctor`, multi-step planning
- **Web UI** — chat, dashboard, cost tracking

The agent has **full PC access** (files, shell, browser, screenshots) when you enable it, and supports **skills**, **memory**, and **multi-agent orchestration**.

## Features

- **Gateway** — WebSocket control plane, sessions, presence
- **Cost tracker** — token counting + session summary
- **Multi-agent** — spawn sub-agents for delegated tasks
- **Skills** — translator, reminders, custom integrations
- **Memory** — MEMORY.md, knowledge graph, persistent context
