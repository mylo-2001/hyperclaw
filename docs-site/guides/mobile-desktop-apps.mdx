## Connect Tab

Mobile (iOS/Android) can connect as "nodes" via the Connect tab. The agent can send commands to nodes (e.g. take photo, get location).

## Desktop Apps

- **macOS menubar** — apps/macos-menubar
- **Electron/Tauri** — optional wrappers for the Web UI

See [docs/mobile-desktop-apps.md](https://github.com/hyperclaw/hyperclaw/blob/main/docs/mobile-desktop-apps.md) for full roadmap.
