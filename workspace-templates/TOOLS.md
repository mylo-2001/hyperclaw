## Skill Configuration

Each skill can be configured in `~/.hyperclaw/openclaw.json` under `skills`.

To install a new skill:
```
hyperclaw hub --install <skill-id>
```

To scan a skill before installing:
```
hyperclaw hub --scan <skill-id>
```
