# Translator Skill

Translate text between languages. No API key required for basic translation — use your language abilities.

## Capabilities

- Translate user-provided text to any supported language
- Detect source language when not specified
- Support for 90+ languages

## Usage

When the user asks to translate:
1. Identify source and target languages
2. Translate accurately, preserving meaning and tone
3. For formal documents, maintain register; for casual chat, adapt accordingly

## Optional: DeepL API

For higher quality, set `DEEPL_API_KEY`. The agent can then use the `translate_deepl` tool if available.
